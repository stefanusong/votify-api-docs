## Get Owned Votes

import Badge from "../../../components/Badge";

<Badge txt="GET" color="green" /> `/api/votes/{id}/answers/mine`

Retrives the vote answers for the current user.\
The user is determined from the JWT Token passed on the authorization header.

### Sample request
Endpoint:
```
http://localhost:8080/api/votes/083fd3b0-1819-46cc-9e9d-d996ad1a03b9/answers/mine
```

Header:
| KEY | VALUE |
| :- | :-|
| Authentication | Bearer eyJhbGciOiJIXVCJ9.eyJpZCI6ImzLWRlMjoi... |
