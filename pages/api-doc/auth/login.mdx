## Login (Get Token)

import Badge from "../../../components/Badge";

<Badge txt="POST" color="blue" /> `/api/auth/login`

Creates a new votify user to create or participate
in any vote.


### Sample request
```json copy
{
    "username": "stefanusong",
    "password": "password"
}
```
| KEY | TYPE | DESCRIPTION |
| :------  | :----- | :----- |
| username | string | The username of the user. |
| password | string | The password of the user. |
