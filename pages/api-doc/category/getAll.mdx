## Get All Categories

import Badge from "../../../components/Badge";

<Badge txt="GET" color="green" /> `/api/categories`

Retrieves all created categories.

### Sample request
Endpoint
```
http://localhost:8080/api/categories
```
Header:
| KEY | VALUE |
| :- | :-|
| Authentication | Bearer eyJhbGciOiJIXVCJ9.eyJpZCI6ImzLWRlMjoi... |
