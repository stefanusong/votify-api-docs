## Get Public Votes

import Badge from "../../../components/Badge";

<Badge txt="GET" color="green" /> `/api/votes/public`

Retrieves all public votes (Votes that have `is_open` attribute set to `true`).

### Sample request
Endpoint
```
http://localhost:8080/api/votes/public
```
Header:
| KEY | VALUE |
| :- | :-|
| Authentication | Bearer eyJhbGciOiJIXVCJ9.eyJpZCI6ImzLWRlMjoi... |
