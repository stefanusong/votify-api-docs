## Get Vote by ID

import Badge from "../../../components/Badge";

<Badge txt="GET" color="green" /> `/api/votes/{id}/questions`

Retrieves vote questions by its vote ID.

### Sample request
Endpoint:
```
http://localhost:8080/api/votes/a8332a0b-8eeb-42e9-9158-390709ff738b/questions
```

Header:
| KEY | VALUE |
| :- | :-|
| Authentication | Bearer eyJhbGciOiJIXVCJ9.eyJpZCI6ImzLWRlMjoi... |
